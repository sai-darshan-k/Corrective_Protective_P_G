Final Constraints & Solutions

Constraint Impact Solution in KrishiPro App
Internet Connectivity High - most features depend on it Cache critical data, SMS backup, offline mode for Daily Log
Sensor Cost High - farmers can't afford full setup Tiered approach: Start with weather, add sensors progressively. FPO group purchases
Technical Literacy Medium - older farmers struggle Voice interface (Krishi AI), vernacular language, simple icons, audio alerts
Smartphone Availability Medium - not all have smartphones Web interface, shared village kiosk, voice-call based alerts
Power Supply Medium - affects sensor operation Solar-powered sensors, battery backup alerts
Data Privacy Low-Medium - farmers may not trust cloud Explain data use, keep personal info local, anonymize for analysis
AI Accuracy Medium - misdiagnosis is risky Confidence scores shown, option to consult expert, continuous training
Chemical Availability High - remote areas have supply issues FPO Connect links to input dealers, advance planning via Daily Log
Labor Shortage High - especially at harvest Early alerts for planning, FPO Connect for shared labor
Market Volatility High - affects harvest timing decisions FPO Connect provides real-time mandi prices, storage options
